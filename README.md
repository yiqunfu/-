# 「轻记」
一款基于 **UniApp** 开发的跨端轻量化应用，整合日记心情记录、待办清单管理、番茄钟专注和数据统计四大核心功能，适配微信小程序、App、H5多平台，专为大学生日常学习生活设计。

## 项目介绍
### 🌟 核心定位
解决大学生时间管理分散、心情记录无专属工具的痛点，打造一款界面简洁、操作便捷、功能聚合的个人管理工具，同时适配多终端使用场景。

### 🎯 核心功能
1. **日记心情记录**：支持文字+图片记录、标签分类（开心/疲惫/学习/日常/emo）、历史检索与心情统计
2.  **待办清单管理**：待办添加、优先级设置、完成状态切换、完成率统计
3.  **番茄钟专注**：25分钟专注+5分钟休息、铃声提醒、专注阶段切换
4.  **数据可视化统计**：待办完成率统计、心情占比饼图、每日心情柱形图、课表式心情展示

### 📱 适配平台
- 微信小程序
- 安卓/iOS App
- H5网页

## 快速开始
### 🔧 开发环境
| 工具/框架 | 版本要求 |
|----------|----------|
| HBuilderX | 3.0+ |
| Node.js | 14.0+ |
| u-charts | 2.0+ |

### 🚀 安装与运行
1. **克隆项目**
    ```bash
    git clone https://github.com/your-username/light-note.git
    ```
2.  **导入项目**
    - 打开 HBuilderX → 点击「文件」→「导入」→ 选择项目目录
3.  **安装依赖插件**
    - 项目右键 →「插件安装」→ 搜索「u-charts」→ 安装官方插件
4.  **配置AppID**
    - 打开 `manifest.json` → 替换 `appid` 为你的 UniApp AppID（微信小程序AppID按需配置）
5.  **运行项目**
    - 项目右键 →「运行到」→ 选择目标平台（微信开发者工具/模拟器/真机）

## 核心技术
1.  **跨端开发**：UniApp 实现一次开发多端发布，减少重复开发成本
2.  **数据存储**：UniApp 本地存储 API（`uni.setStorageSync`/`uni.getStorageSync`）实现数据持久化
3.  **数据可视化**：u-charts 插件实现饼图、柱形图等图表展示
4.  **组件化开发**：Vue 组件化思想，封装可复用的待办项、日记卡片组件
5.  **样式管理**：SCSS 全局变量统一管理主题色、字体大小等样式属性
